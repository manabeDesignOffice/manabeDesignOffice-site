# MANABE Design Office - Single Page Site (GitHub Pages)

## 使い方（最短）
1. `index.html`, `style.css`, `script.js` を GitHub のリポジトリ直下にアップロード
2. ロゴ画像を `logo.png` というファイル名でリポジトリ直下に置く（PNG推奨）
3. GitHub の Settings → Pages → Deploy from branch → main / (root) を選択
4. 表示された URL で公開
