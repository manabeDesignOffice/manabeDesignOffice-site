# MANABE Design Office - Single Page Site (GitHub Pages)

## 使い方（最短）
1. このフォルダの `index.html`, `style.css`, `script.js` を GitHub のリポジトリ直下にアップロード
2. GitHub の Settings → Pages → Deploy from branch → main / (root) を選択
3. 表示された URL で公開

## 置き換えポイント
- `YOUR_X_HANDLE` をあなたの X ユーザー名に置き換えてください（@なし）
- `you@example.com` を連絡先に置き換えてください

## ファイル構成
- index.html : 本体
- style.css  : スタイル
- script.js  : ハンバーガー（モバイルメニュー）と小さな処理
